```json
{
  "trading": {
    "symbol": "BTCUSDT",
    "risk_per_trade": 0.005,
    "max_leverage": 20
  }
}
```
**详细说明**: [配置参数说明](CONFIGURATION.md#核心交易配置)

### 🔍 MACD配置
```json
{
  "macd_divergence": {
    "lookback_period": 100,
    "min_peak_distance": 5,
    "prominence_mult": 0.5
  }
}
```
**详细说明**: [配置参数说明](CONFIGURATION.md#MACD背离配置)

```json
  }
}
```
